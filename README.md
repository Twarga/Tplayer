# Tplayer

A beautiful, Spotify-like local music player for Linux with YouTube download integration.

## Features

- Local music library with automatic scanning
- YouTube download via yt-dlp
- Beautiful dark/light themes (warm amber accent)
- Three-column layout: sidebar, main content, now playing panel
- Equalizer, playlists, queue management
- Last.fm scrobbling
- MPRIS media keys support
- Mini player mode

## Tech Stack

- Electron + Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- better-sqlite3 for local database
- FFmpeg for audio decoding
- Zustand for state management
