Major YouTube Import overhaul with playlist support, advanced download settings, UI redesign, and auto-playlist creation.

## New Types (Draft)

```ts
export interface PlaylistVideoInfo {
  videoId: string
  title: string
  channel: string
  duration: string
  thumbnail: string
}

export interface DownloadOptions {
  audioFormat: 'mp3' | 'm4a' | 'flac' | 'wav' | 'opus'
  audioQuality: number // 0 = best, 10 = worst
  embedThumbnail: boolean
  addMetadata: boolean
  outputDir?: string
}

export interface BatchDownloadProgressPayload {
  playlistUrl: string
  completed: number
  total: number
  currentVideoId: string
  currentTitle: string
  currentProgress: number
}
```

## Notes
- Keep existing single-video download backward-compatible
- Playlist downloads should be sequential to avoid overwhelming yt-dlp / network
- Auto-playlist creation requires `api.playlist.create` + `api.playlist.addTracks`
- UI must remain responsive during batch downloads (progress via IPC events)
