Local-first desktop music player

Your library, imported tracks, and queue in one focused player.

Tplayer turns local folders into a refined daily music player with fast playback, YouTube import, Last.fm scrobbling, MPRIS controls, EQ, and a warm dark interface.

Download latest View source

Linux AppImage first. Windows builds are produced by the release workflow.

Built for people who want their own music collection to feel immediate, elegant, and alive. No streaming clone framing. No filler dashboard. Just library, import, queue, and playback.

Interface

Dark, editorial, and built around album art.

Tplayer uses a warm ink-and-gold palette, large artwork, thin dividers, and direct controls. The UI is designed to feel like a music product instead of a generic Electron panel stack.

Workflow

From folder to daily player.

Add local folders, import missing tracks, build a queue, tune the EQ, and keep listening history synced through Last.fm. Linux desktop controls work through MPRIS.

  1. 01 Scan local music folders
  2. 02 Import audio from YouTube
  3. 03 Queue, seek, shuffle, and repeat
  4. 04 Scrobble and control from the desktop

Release

Installable builds live on GitHub Releases.

The release workflow produces a Linux AppImage and a Windows executable when a version tag is pushed.
